استخراج‌کننده صدا از یوتیوب

استخراج صدا از ویدیوها و پلی‌لیست‌های یوتیوب — رایگان، متن‌باز، و ساده. با رابط وب زیبا و ابزار خط فرمان قدرتمند.

دانلود آخرین نسخه

ویژگی‌ها

  • رابط وب — رابط Flask زیبا، نیازی به ترمینال نیست
  • CLI — ابزار خط فرمان کامل برای کاربران پیشرفته
  • کنترل بیت‌ریت — کیفیت صدای دلخواه خود را انتخاب کنید
  • تقسیم بر اساس فصل — تقسیم خودکار صدا بر اساس فصل‌های یوتیوب
  • پشتیبانی از پلی‌لیست — استخراج کامل پلی‌لیست‌ها با یک دستور
  • چند‌پلتفرمی — کار می‌کند روی Linux، macOS و Windows

شروع سریع

  1. نصب وابستگی‌ها 
    pip install -r requirements.txt
  2. راه‌اندازی رابط وب 
    ./start_ui.sh
  3. باز کردن مرورگر در آدرس http://localhost:5000
  4. لینک یوتیوب را وارد کنید و دکمه دانلود را بزنید!
