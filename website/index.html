YouTube Audio Extractor

Extract audio from YouTube videos and playlists — free, open-source, and easy to use. Comes with a beautiful web UI and a powerful CLI.

Download Latest

Features

  • Web UI — Beautiful Flask-based interface, no terminal needed
  • CLI — Full-featured command-line tool for advanced users
  • Bitrate control — Choose your audio quality
  • Chapter splitting — Automatically split audio by YouTube chapters
  • Playlist support — Extract entire playlists in one command
  • Cross-platform — Works on Linux, macOS, and Windows

Quick Start

  1. Install dependencies 
    pip install -r requirements.txt
  2. Start the web UI 
    ./start_ui.sh
  3. Open your browser at http://localhost:5000
  4. Paste a YouTube URL and click Download!
